# dashboard-hse-offshore
Dashboard interativo de performance de segurança (HSE) desenvolvido em Power BI.
![Visualização do Dashboard](dashboard.png)
## 📊 Sobre o Projeto
Este dashboard de **HSE Analytics** foi desenvolvido para monitorar indicadores críticos de segurança em operações industriais/offshore.

### 🛠️ Tecnologias Utilizadas
* **Power BI**: Criação de visualizações e dashboards interativos.
* **Power Query (ETL)**: Tratamento e limpeza de dados de incidentes.
* **DAX**: Cálculo de medidas complexas como Taxa de Frequência e Gravidade.

### 📈 Principais Insights
* Monitoramento de conformidade com a **NR-37**.
* Análise de tendência de incidentes por mês e turno.
* Identificação dos principais agentes causadores (Cortes, Queimaduras, etc.).
